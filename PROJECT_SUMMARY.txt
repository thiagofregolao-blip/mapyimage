PROJECT OVERVIEW
================================================================================

Mapy Image Manager is a complete, production-ready web application for managing
product images in large e-commerce catalogs (40,000+ products).

MAIN FEATURES:
✓ Dashboard with product grid and statistics
✓ Excel (XLSX) import/export functionality
✓ Automated image search via Google Images (SerpAPI)
✓ Individual and batch image search
✓ Product filtering and search
✓ Image status tracking (Complete, Partial, Pending, No Image)
✓ SQLite database with optimized indexes
✓ Spanish language interface
✓ Responsive design with TailwindCSS
✓ Production-ready error handling and logging

DATABASE SCHEMA
================================================================================

TABLE: products

Columns:
  id              INTEGER PRIMARY KEY
  sku             TEXT UNIQUE (indexed)
  descripcion_original  TEXT
  marca           TEXT (indexed)
  subcategoria    TEXT
  categoria       TEXT (indexed)
  nombre_es       TEXT (indexed)
  nome_pt         TEXT
  desc_es         TEXT
  desc_pt         TEXT
  preco           REAL
  image_url_1     TEXT
  image_url_2     TEXT
  image_status    TEXT (indexed) - pending/partial/complete/no_image
  search_query    TEXT
  created_at      TIMESTAMP
  updated_at      TIMESTAMP

Indexes:
- idx_sku (for fast SKU lookups)
- idx_categoria (for category filtering)
- idx_marca (for brand filtering)
- idx_status (for status filtering)
- idx_search (for search queries)

Performance: Optimized for 40,000+ products with fast pagination

================================================================================
INSTALLATION & SETUP
================================================================================

QUICK START:

1. Windows: Double-click run.bat
2. Linux/Mac: ./run.sh
3. Docker: docker-compose up

MANUAL SETUP:

1. Create virtual environment:
   python -m venv venv
   source venv/bin/activate  # Windows: venv\Scripts\activate

2. Install dependencies:
   pip install -r requirements.txt

3. Setup SerpAPI (required for image search):
   - Register at https://serpapi.com
   - Copy API key
   - Create .env file with: SERPAPI_KEY=your_key

4. Start application:
   python app.py
   # or: uvicorn app:app --reload

5. Open browser:
   http://localhost:8000

DOCKER:

1. Create .env file with SERPAPI_KEY
2. Run: docker-compose up
3. Open: http://localhost:8000

See QUICKSTART.md and INSTALL.md for detailed instructions.

EXCEL IMPORT FORMAT
================================================================================

Required Columns (in any order):
- SKU                           (Unique product identifier)
- MARCA                         (Brand)
- CATEGORIA                     (Main category)
- NOMBRE_ESTANDAR_ES           (Product name in Spanish)
- PRECO                        (Price)

Optional Columns:
- SUBCATEGORIA                 (Subcategory)
- NOMBRE_PADRONIZADO_PT        (Product name in Portuguese)
- DESC_ECOMMERCE_ES            (Description in Spanish)
- DESC_ECOMMERCE_PT            (Description in Portuguese)
- URL_IMAGEN                   (Existing image URL)

The system automatically detects column names with flexible matching.
Example: "NOMBRE_ESTANDAR_ES", "NOMBRE ESTANDAR ES", "NOMBRE_ES" all work.
